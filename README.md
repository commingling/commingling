# commingling | Java Backend Engineer

Java 后端工程师，关注支付系统、DDD、分布式事务和高并发稳定性建设。

## About Me

- 聚焦 Java 后端、分布式系统、支付链路和工程稳定性。
- 熟悉 Spring Boot、MySQL、Redis、消息队列、Linux 和常见 CI/CD 流程。
- 重视业务建模、边界划分、可观测性、故障恢复和性能优化。
- 喜欢把复杂业务拆成清晰、可维护、可演进的工程方案。

## Focus

- Payment Systems: 支付链路、订单状态流转、对账、幂等和异常补偿。
- Domain-Driven Design: 业务边界、聚合设计、应用服务编排和领域模型沉淀。
- Distributed Transactions: 本地事务、最终一致性、消息可靠投递和补偿机制。
- High-Concurrency Optimization: 缓存、限流、异步化、数据库索引和热点隔离。
- Engineering Quality: 日志、监控、告警、自动化部署和线上问题复盘。

## Interview Keywords

- 复杂业务建模: 从需求边界、状态机、领域对象和应用服务编排落地业务。
- 支付稳定性: 幂等、防重、补偿、对账、异常订单恢复和资金链路风险控制。
- 性能优化: 慢 SQL 分析、索引设计、缓存策略、异步削峰和热点隔离。
- 线上治理: 日志规范、核心指标、告警分级、问题定位和复盘改进。
- 工程协作: 代码评审、接口契约、发布流程、灰度验证和文档沉淀。
